      ☁️ Cloud Sync (Firebase)

      เชื่อมต่อ Firebase เพื่อ collaborate กับทีม (ฟรี ไม่จำกัด user) • หากไม่ตั้งค่า ระบบจะใช้ localStorage ในเครื่องเท่านั้น

      📖 วิธีตั้งค่า Firebase (ฟรี)
      1. เข้า console.firebase.google.com → ล็อกอินด้วย Google
      2. กด Add project → ตั้งชื่อ → ปิด Analytics ก็ได้ → Create
      3. ในหน้า project ดูที่ Project Overview → กดไอคอน </> (Web app)
      4. ตั้งชื่อ app → Register → copy เฉพาะส่วน firebaseConfig (รูปแบบ JSON)
      5. เมนูซ้าย → Build → Firestore Database → Create database → Start in test mode → เลือก region (asia-southeast1)
      6. กลับมาวาง config ในช่องด้านบน → กด บันทึก & เชื่อมต่อ
      7. (แนะนำ) ตั้ง Firestore Security Rules ตามที่ระบบ generate ให้ — กดปุ่ม "Copy Security Rules"
